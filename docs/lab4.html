We used the OV7670 camera datasheet http://web.mit.edu/6.111/www/f2016/tools/OV7670_2006.pdf to write registers to:

- Reset all registers: COM7
- Use the external clock as internal clock: CLKRC
- Set the camera to output the resolution and pixel format RGB 565, QCIF: COM7, COM15
- Enable a color bar test: COM7, COM17
- Vertical and mirror flip the output image: MVFP
- Fix the gain and other parameters (freeze AGC/AEC): COM9
- Enable scaling: COM3

The I2C address uses the least significant bit to distinguish between read and write and is 0x21.
